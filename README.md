## Установка и запуск

### 1. Установить зависимости фронтенда

```bash
npm install
```

### 2. Запустить backend

Backend находится в папке `backend-source`.

```bash
cd backend-source
npm install
npm run dev
```

По умолчанию backend запускается на:

```text
http://localhost:3333
```

### 3. Запустить frontend

Из корня проекта:

```bash
npm run dev
```

Frontend запускается на:

```text
http://localhost:5173
```

## Сборка проекта

```bash
npm run build
```
